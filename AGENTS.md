# 템플릿 갱신 방법

- https://www.voidtools.com/Template.zip 다운로드
- 압축 해제 후 적절한 위치에 추가 또는 덮어쓰기

# 우선 순위

- 실사용 시 가장 오해없이 자연스러운 표현을 지향합니다
- 용어집 용어를 선호합니다
- 간결한 표현을 선호합니다

# 번역 톤

- '~의'가 필요하다면
  - 생략 가능한지 확인합니다.
  - 자연스러운 다른 표현이 없다면 사용합니다.
- 피동보다 주동 표현을 선호합니다.
- ~니다, ~까요?, ~세요 정도의 존대를 사용합니다.
  - 라이선스같은 외부 텍스트는 보존합니다.
