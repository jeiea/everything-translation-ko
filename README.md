# Everything 한국어 번역

[Everything](https://www.voidtools.com/) 파일 검색 프로그램의 한국어 번역 저장소입니다.

질문/의견은 이슈나 [포스트 댓글](https://ddwroom.tistory.com/33)로 남겨주세요.

## 사용법

apply.cmd로 Korean.txt에서 번역 파일을 생성하고 Everything(혹은 Everything 알파 버전)에 적용합니다.

dev.cmd를 실행해서 [WinMerge](https://winmerge.org/?lang=ko)로 템플릿과 번역 파일을 비교하면서 편집할 수 있습니다.
