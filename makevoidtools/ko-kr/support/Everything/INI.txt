==Everything으로 설정 보고 변경하기==
[[support/everything/options|Everything 설정]] 창에서 설정 대부분을 바꿀 수 있습니다.

특정 설정의 현재값을 알고 싶다면 검색 상자에 다음 내용을 입력하고 엔터를 쳐보세요.
<pre>
/<ini 옵션>
</pre>
<ini 옵션>은 아래 ini 옵션을 넣으면 되고, 모든 옵션을 지원하지는 않습니다.

예시로 '''snap''' 옵션의 값을 알고 싶다면 아래를 입력하고 엔터를 칩니다.
<pre>
/snap
</pre>

현재 설정 값은 3초간 상태 표시줄에 표시됩니다.


설정 값을 바꾸려면 아래를 입력하고 엔터를 칩니다.
<pre>
/<ini 옵션>=<값>
</pre>
아래 ini 옵션을 넣으면 되고 <값>은 새 설정 값입니다. 모든 옵션을 지원하진 않습니다.

예시로 '''snap''' 설정을 켜려면 아래를 입력하고 엔터를 칩니다.
<pre>
/snap=1
</pre>

새 설정 값은 3초간 상태 표시줄에 표시됩니다.

==Everything.ini 편집하기==
Everything.ini는 아래 경로에 있습니다.
<pre>
%APPDATA%\Everything
</pre>
[[support/everything/options#%APPDATA%\Everything 폴더에 설정 저장|%APPDATA%\Everything 폴더에 설정 저장]]을 껐다면 Everything.exe가 있는 경로에 있을 겁니다.

[[support/everything/options#%APPDATA%\Everything 폴더에 설정 저장|%APPDATA%\Everything 폴더에 설정 저장]] 설정은 Everything.exe가 있는 곳의 Everything.ini에 저장됩니다.

현재 사용 중인 Everything.ini 파일을 열려면, 아래를 입력하고 엔터를 치세요.:
<pre>
about:config
</pre>

'''Everything.ini는 자동 생성됩니다. Everything 실행 도중 Everything.ini를 바꿨더라도 Everything 종료 시 덮어씌워집니다.'''

Everything.ini를 바꾸기 전에 ''Everything''을 꺼 주세요.

Everything을 끄려면
<ul>
*Everything 트레이 아이콘을 우클릭하고 종료를 클릭합니다.
</ul>

UTF-8로 Everything.ini를 작성해야 합니다.

==구문==
<pre>
[섹션]
키=값
</pre>

==[Everything]==
<pre>
[Everything]
</pre>
위 텍스트로 Everything 섹션을 시작합니다.
사실 이 섹션말고 안 씁니다.

==일반==
Everything 일반 설정입니다.

<dl>
;''app_data''
설정과 자료 위치입니다. 1로 설정하면 %APPDATA%\Everything에 저장합니다. 0이면 Everything.exe가 있는 곳에 저장합니다.
이 옵션은 Everything.exe가 있는 곳의 Everything.ini에 항상 저장됩니다.

;''run_as_admin''
Everything을 관리자 권한으로 실행합니다. 1이면 일반 계정 권한으로 실행할 때 관리자 권한으로 재실행합니다. 관리자 권한을 가지고 있었다면 그대로 진행합니다. 0이면 관리자 권한으로 재실행하지 않습니다.
이 옵션은 Everything.exe가 있는 곳의 Everything.ini에 항상 저장됩니다.

;''allow_multiple_windows''
Everything.exe를 실행할 때 이미 실행 중인 Everything의 기존 검색 창을 보여줄지 새 창을 보여줄지 정합니다. 1이면 새 창을 항상 만듭니다. 0이고 기존 창이 없으면 새로 만듭니다. 0이고 기존 창이 있으면 그 창을 띄워줍니다.

;''run_in_background''
모든 검색 창을 껐을 때 Everything을 종료해야하는지 정합니다. 1이면 창을 끄더라도 프로그랭은 남아있습니다. 0이면 모든 창을 껐을 때 프로그랭이 종료됩니다.

;''show_tray_icon''
트레이 영역에 Everything 아이콘을 보여줄지 정합니다. 1이면 트레이 아이콘을 보여주고, 0이면 안 보여줍니다.

;''language''
Everything 창 언어의 십진법 [[support/everything/language ids|언어 ID]]입니다. 1042면 한국어를 사용합니다. 0으면 사용자 기본 언어를 씁니다.

;''open_folder_command2''
;''open_file_command2''
;''open_path_command2''
;''explore_command2''
;''explore_path_command2''
폴더, 파일, 경로를 열거나 탐색할 때 사용할 명령어입니다.
자세한 건 [[support/everything/customizing#외부_파일_관리자|외부 파일 관리자]]를 참고하세요.

;''last_export_type''
마지막으로 내보낸 형식입니다. 다음 중 하나를 쓸 수 있습니다.
{|
! 종류
! 설명
|-
| 0
| EFU 파일
|-
| 1
| CSV 파일
|-
| 2
| TXT 파일
|}

;''check_for_updates_on_startup''
Everything을 시작할 때 업데이트 확인을 합니다. 1이면 켜고 0이면 끕니다.

;''connect_history_hosts''
성공한 ETP 연결 히스토리입니다.

;''connect_history_ports''
성공한 ETP 연결 히스토리 각각의 포트 목록입니다.

;''connect_history_usernames''
성공한 ETP 연결 히스토리 각각의 사용자 ID 목록입니다.

;''connect_history_link_types''
성공한 ETP 연결 히스토리 각각의 연결 종류입니다. 다음 중 하나를 쓸 수 있습니다.
{|
! 종류
! 설명
|-
| 0
| C:\ 형식.
|-
| 1
| \\server\C 형식.
|-
| 2
| \\server\$C 형식.
|-
| 3
| ftp://server/C 형식.
|}

;''search_history_enabled''
검색어 기록을 켜거나 끕니다. 1이면 켜고, 0이면 끕니다.

;''run_history_enabled''
실행 항목 기록을 켜거나 끕니다. 1이면 켜고, 0이면 끕니다.

;''search_history_days_to_keep''
검색어를 보관할 최대 일수입니다.

;''run_history_days_to_keep''
실행 기록을 보관할 최대 일수입니다.

;''search_history_keep_forever''
검색 기록을 영구히 보관합니다.
1이면 켜고, 0이면 끕니다.

;''run_history_keep_forever''
실행 기록을 영구히 보관합니다.
1이면 켜고, 0이면 끕니다.

;''search_history_always_suggest''
검색어를 타자를 칠 때마다 제안할 지 Ctrl + 스페이스 키를 누를때만 제안할 지 정합니다. 1이면 항상 제안하고, 0이면 Ctrl + 스페이스를 눌러야 제안합니다.

;''search_history_sort''
검색 기록 순서를 정합니다.
다음 중 하나를 쓸 수 있습니다.
{|
! 값
! 설명
|-
| 0
| 이름순 정렬
|-
| 1
| 검색 횟수순 정렬
|-
| 2
| 검색 날짜순 정렬
|}

;''search_history_sort_ascending''
검색 기록 순서를 오름차순으로 할지 정합니다.
1이면 오름차순, 0이면 내림차순입니다.

;''search_history_x''
;''search_history_y''
검색 히스토리 창 위치입니다.

;''search_history_wide''
;''search_history_high''
검색 히스토리 창 크기입니다.

;''search_history_column_search_wide''
히스토리 창 검색어 열 픽셀 너비입니다.

;''search_history_column_search_order''
히스토리 창 검색어 열 위치 순서입니다.

;''search_history_column_count_wide''
히스토리 창 검색 횟수 열 픽셀 너비입니다.

;''search_history_column_count_order''
히스토리 창 검색 횟수 열 위치 순서입니다.

;''search_history_column_date_wide''
히스토리 창 검색 날짜 열 픽셀 너비입니다.

;''search_history_column_date_order''
히스토리 창 검색 날짜 열 위치 순서입니다.
</dl>

==검색 창==
외관 관련 설정입니다.

<dl>
;''window_x''
마지막으로 끈 검색 창 왼쪽 위 귀퉁이의 화면 픽셀 기준 가로 좌표입니다. 새 검색 창을 이 값 위치에 띄웁니다.

;''window_y''
마지막으로 끈 검색 창 왼쪽 위 귀퉁이의 화면 픽셀 기준 세로 좌표입니다. 새 검색 창을 이 값 위치에 띄웁니다.

;''window_wide''
;''window_high''
마지막으로 끈 Everything 검색 창의 너비, 높이 픽셀 수 입니다. 새 검색 창을 이 크기로 띄웁니다.

;''maximized''
1이면 새 검색 창을 최대화된 상태로 띄웁니다. 0이면 최대화하지 않습니다.

;''minimized''
1이면 새 검색 창을 최소화된 상태로 띄웁니다. 0이면 최소화하지 않습니다.

;''fullscreen''
1이면 검색 창을 전체 화면으로 보여줍니다. 0이면 일반 창으로 보여줍니다.

;''ontop''
항상 위에 기능입니다. 0이면 일반 창이고, 1이면 항상 위에 창을 고정합니다. 2면 검색 도중에만 항상 위에 고정합니다.

;''alternate_row_color''
1이면 검색 결과 배경색으로 두 색을 교대로 사용합니다. 0이면 한 색깔만 사용합니다.

;''show_mouseover''
포인터를 갖다댄 검색 결과 행을 다른 색으로 강조합니다. 1이면 다른 색을 쓰고, 0이면 기본 색을 씁니다.

;''show_highlighted_search_terms''
검색어 일치 부분을 강조합니다. 1이면 켜고, 0이면 끕니다.

;''text_size''
검색 결과 글꼴 픽셀 높이 오프셋입니다. 기존 설정에 이 값을 더하므로 0이 기본값입니다.

;''hide_empty_search_results''
1이면 검색 상자가 비어있을 때 아무 결과도 보여주지 않습니다. 0이면 모든 결과를 보여줍니다.

;''clear_selection_on_search''
1이면 검색어가 바뀔 때 선택 영역을 초기화합니다. 0이면 남아있는 항목 선택을 유지합니다.

;''show_selected_item_in_statusbar''
1이면 현재 선택한 항목 정보를 상태 표시줄에 표시합니다. 0이면 선택한 항목 개수를 알려줍니다.

;''translucent_selection_rectangle_alpha''
드래그 선택 사각형의 투명도입니다. 0 = 투명 ~ 255 = 불투명까지 쓸 수 있습니다.

;''last_options_page''
마지막 Everything 설정 페이지입니다. 설정 창을 띄울 때 이 페이지를 보여줍니다.

;''invert_layout''
LTR/RTL을 전환합니다. 1이면 전환합니다.

;''allow_multiple_windows_from_tray''
1이면 트레이 아이콘을 더블 클릭 했을 때 무조건 새 검색 창을 띄웁니다. 0이면 기존 창이 있을 땐 기존 창을 보여줍니다.

;''single_click_tray''
1이면 트레이 아이콘을 클릭해서 검색 창을 열 수 있습니다. 0이면 더블 클릭으로 열 수 있습니다.

;''close_on_execute''
1이면 항목을 실행했을 때 검색 창을 닫습니다. 0이면 창을 남겨둡니다.

;''double_click_path''
1이면 검색 항목의 경로 칸을 더블 클릭해서 경로를 열 수 있습니다. 0이면 더블 클릭을 무시합니다.

;''update_display_after_scroll''
1이면 스크롤한 직후에 다시 그립니다. 0이면 영역 무효화만 하고 갱신은 미룹니다.

;''auto_scroll_view''
1이면 드래그 선택 포인터가 화면 밖으로 나갔을 때 자동으로 스크롤해서 선택 영역을 확장해 줍니다. 0이면 포인터를 움직일 때만 스크롤해줍니다.

;''double_quote_copy_as_path''
1이면 경로를 복사할 때 "로 감싸줍니다. 0이면 경로만 복사합니다.

;''rename_select_filepart_only''
1이면 이름 바꾸기 시 확장자를 제외하고 선택합니다. 0이면 확장자를 포함해서 선택합니다.

;''select_search_on_mouse_click''
검색 상자를 선택할 때 검색어 전체를 선택합니다.

;''focus_search_on_activate''
검색 창을 선택하면 자동으로 검색 상자에 초점을 맞춥니다.

;''single_click_open''
검색 항목을 여는 방법을 정합니다. 0이면 시스템 기본 설정을 씁니다. 1이면 클릭으로 항목을 엽니다. 2면 더블 클릭으로 항목을 엽니다.

;''underline_icon_titles''
검색 결과 이름에 밑줄을 칩니다. [[#single_click_open|클릭으로 항목을 실행]]해야 적용됩니다.
0이면 시스템 설정을 씁니다. 1이면 항상 밑줄을 칩니다. 2면 밑줄치지 않습니다. 3이면 포인터가 해당 이름 위에 있을 때 밑줄을 칩니다.

;''wrap_focus''
결과 목록 양 끝을 방향키로 이동할 때의 설정입니다. 다음 중 하나를 쓸 수 있습니다.
{|
! 값
! 설명
|-
| 0
| 결과 목록 맨 처음과 맨 마지막이 연결되어있지 않습니다.
|-
| 1
| 결과 목록 맨 처음과 맨 마지막이 연결되어 있습니다.
|-
| 2
| 결과 목록 맨 처음과 맨 마지막 사이에 검색 상자가 들어가 연결되어 있습니다.
|}

;''load_icon_priority''
;''load_thumbnail_priority''
;''load_fileinfo_priority''
불러오기 작업 스레드 우선순위입니다.
다음 중 하나를 쓸 수 있습니다.
{|
! 값
! 설명
|-
| -15
| 매우 낮음.
|-

|-
| 0
| 보통.

;''full_row_select''
1이면 행 어디를 클릭하더라도 행을 선택할 수 있고 행 전체를 선택 영역으로 표시합니다. 0이면 반드시 이름을 클릭해야 합니다.

;''double_buffer''
1이면 더블 버퍼링을 켜고 0이면 끕니다. 더블 버퍼링은 화면 깜박임을 줄이는데 도움을 주지만 성능이 매우 안 좋은 컴퓨터에서는 끄는 게 도움이 될 수 있습니다.

1이면 항목을 선택했을 때도 전체 항목 개수를 보여줍니다. 0이면 선택 항목 개수만 보여줍니다.

0이면 머리 행을 클릭해 날짜순 정렬할 때 새 것부터 보여줍니다. 2면 오래된 것부터 보여줍니다.

0이면 머리 행을 클릭해 크기순 정렬할 때 큰 것부터 보여줍니다. 2면 작은 것부터 보여줍니다.

;''show_size_in_statusbar''

==숨겨진 설정==
여기 설정은 Everything UI에서 설정할 수 없습니다.
Everything.ini를 직접 편집해야만 합니다.

<dl>
;''allow_http_server''
1이면 HTTP 서버를 허용하고 0이면 HTTP 서버를 끄는 동시에 GUI로 켤 수 없게 합니다.
Everything.exe 위치에 있는 Everything.ini에서만 설정할 수 있습니다.

;''allow_etp_server''
1이면 ETP 서버를 허용하고 0이면 ETP 서버를 끄는 동시에 GUI로 켤 수 없게 합니다.
Everything.exe 위치에 있는 Everything.ini에서만 설정할 수 있습니다.

;''allow_multiple_instances''
1이면 실행할 때마다 새 Everything 프로세스를 생성합니다 (이거 대신 [[support/everything/Multiple instances|유명 인스턴스]]를 권장합니다). 0이고 기존 프로세스가 있으면 명령어를 기존 프로세스에 전달합니다. 0이고 기존 프로세스가 없으면 새로 실행합니다.

==백업==
모든 설정을 백업하려면
<ul>
*'''Everything'''에서 아래 명령어를 입력하고 엔터를 칩니다.
<pre>
/config_save
</pre>
*파일 이름을 정하고 '''저장'''을 클릭합니다.
</ul>

==복원==
모든 설정을 복원하려면
<ul>
*'''Everything'''에서 아래 명령어를 입력하고 엔터를 칩니다.
<pre>
/config_load
</pre>
*Everything.ini를 선택하고 '''열기'''를 클릭합니다.
</ul>

==기본 설정 복원==
모든 설정을 기본값으로 되돌리려면
<ul>
* ''Everything''을 종료합니다.
* %APPDATA%\Everything\Everything.ini를 삭제합니다.
* Everything을 재시작합니다.
</ul>

==다른 설정 경로==
Everything.exe [[support/everything/command line options#-config|-config]] <Everything.ini>로 설정 파일 위치를 지정할 수 있습니다.
