=Everything 제거하기=
Everything을 설치할 때 제거실행파일(언인스톨러)도 생성합니다.

언인스톨러로 Everything을 완전히 제거하려면
<ul>
*'''제어판'''에서 '''프로그램 및 기능'''을 엽니다.
*'''Everything'''을 우클릭합니다.
*'''제거/변경'''을 선택합니다.
*Everything 다국어 버전을 설치했다면 언인스톨러 언어를 선택하고 '''OK'''를 클릭합니다.
*'''제거'''를 클릭합니다.
</ul>

휴대 버전 Everything을 완전히 제거하라면
<ul>
*명령 프롬프트에서 아래 명령어를 실행합니다.
<pre>
Everything.exe -uninstall
</pre>
*Everything.exe를 삭제합니다.
</ul>

==시스템 변경==
아래 설정은 시스템을 수정합니다.
<ul>
*%APPDATA%\\Everything 폴더에 설정 저장
*시스템 시작 시 Everything 실행
*Everything 서비스
*탐색기에서 우클릭 메뉴 보여주기
*시작 메뉴 바로가기
*바탕화면 바로가기
*빠른 실행 바로가기
*URL 프로토콜
*EFU 파일 연결
</ul>

위 설정을 변경하려면
<ul>
*'''Everything''' - '''도구''' 메뉴 - '''설정'''을 클릭합니다.
*'''일반''' 탭을 클릭합니다.
</ul>

Everything을 제거할 때 모든 Everything 시스템 변경사항도 제거합니다.

==Everything이 생성하는 파일==
Everything이 아래 파일을 %APPDATA%\Everything에 만들 수 있습니다.
<ul>
*Everything.db
*Everything.db.tmp
*Everything.ini
*Everything.ini.tmp
*Errorlog.txt
*Bookmarks.csv
*Bookmarks.csv.tmp
*Filters.csv
*Filters.csv.tmp
*Search History.csv
*Search History.csv.tmp
*Run History.csv
*Run History.csv.tmp
*Logs\HTTP_Server_Log.txt
*Logs\ETP_Server_Log.txt
</ul>

이 파일들은 Everything을 제거할 때 같이 제거됩니다.
[[support/everything/options#%APPDATA%\Everything 폴더에 설정 저장|%APPDATA%\Everything 폴더에 설정 저장]]을 끄면 이 파일들은 Everything.exe와 같은 경로에 저장됩니다.
[[support/everything/multiple instances|다중 인스턴스]]를 쓰고 있다면 위 파일 이름이 달라집니다.
[[support/everything/multiple instances#유명 인스턴스|유명 인스턴스]]를 사용했었고 그걸 제거하려고 한다면, 각 인스턴스를 Everything.exe -uninstall -instance <인스턴스 이름>으로 일일히 제거해야 합니다.

==수동 제거==
<ul>
*Everything과 Everything 서비스를 종료합니다.
*C:\Program Files\Everything 폴더를 삭제합니다.
*C:\Program Files (x86)\Everything 폴더를 삭제합니다.
*%APPDATA%\Everything 폴더를 삭제합니다.
*HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Run에 있는 Everything 레지스트리 값을 삭제합니다.
*HKEY_LOCAL_MACHINE\SOFTWARE\Wow6432Node\Microsoft\Windows\CurrentVersion\Run에 있는 Everything 레지스트리 값을 삭제합니다.
*HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\Everything 레지스트리 키를 삭제합니다.
*HKEY_CLASSES_ROOT\Folder\shell\Search Everything... 레지스트리 키를 삭제합니다.
*%APPDATA%\Microsoft\Windows\Start Menu\Programs\Everything 폴더를 삭제합니다.
*%USERPROFILE%\Start Menu\Programs\Everything 폴더를 삭제합니다.
*%USERPROFILE%\Desktop\Search Everything.lnk 바로가기를 삭제합니다.
*%APPDATA%\Microsoft\Internet Explorer\Quick Launch\Search Everything.lnk 바로가기를 삭제합니다.
*HKEY_CLASSES_ROOT\ES 레지스트리 키를 삭제합니다.
*HKEY_CLASSES_ROOT\.efu 레지스트리 키를 삭제합니다.
*HKEY_CLASSES_ROOT\Everything.FileList 레지스트리 키를 삭제합니다.
*HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Uninstall\Everything 레지스트리 키를 삭제합니다.
*HKEY_LOCAL_MACHINE\SOFTWARE\Wow6432Node\Microsoft\Windows\CurrentVersion\Uninstall\Everything 레지스트리 키를 삭제합니다.
</ul>
