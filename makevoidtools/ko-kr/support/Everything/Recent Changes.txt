=최근 변경 목록=
최근 변경 목록은 Everything을 시작한 이후 이름 변경을 포함해 변경된 파일 및 폴더 목록입니다.

==최근 변경 목록 검색==
rc:나 recentchange:를 사용해 최근 변경 목록을 검색합니다.
<pre>
rc:<날짜>
</pre>

오늘자 변경 항목을 찾으려면
<pre>
rc:today
</pre>

어제자 변경 항목을 찾으려면
<pre>
rc:yesterday
</pre>

지난 2시간 동안의 변경 항목을 찾으려면
<pre>
rc:last2hours
</pre>

지난 10분 동안의 변경 항목을 찾으려면
<pre>
rc:last10mins
</pre>

모든 rc: 검색 문법은 [[support/everything/searching#함수 구문|검색]]을 참고하세요.

==USN 저널을 최근 변경 목록에 포함하기==
기본적으로 Everything은 실행 후의 변경 사항만 보관합니다. USN 변경 저널을 최근 변경 사항에 포함하면 더 오래전의 변경 사항까지 빨리 검색할 수 있을 겁니다.

USN 저널을 최근 변경목록에 포함하려면
<ul>
*'''Everything''' - '''도구''' 메뉴 - '''설정'''을 클릭합니다.
*'''NTFS''' 탭을 클릭합니다.
*각 로컬 NTFS 볼륭에서:
<ul>
*'''USN 저널에서 최근 변경사항 가져오기'''를 체크합니다.
</ul>
*'''확인'''을 클릭합니다.
</ul>

==최근 변경 날짜 정렬==
최근 변경 날짜로 정렬하는 건 빠르고 지속적입니다.
결과 목록의 빈 곳을 우클릭하고 , 정렬 메뉴의 최근 변경 날짜를 클릭합니다.
Everything 색인 변경을 실시간으로 보려면 최근 변경 날짜로 정렬하세요.
